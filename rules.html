General Clan Rules

  • No homophobic, racist, or otherwise derogatory language.
  • We don't do donation ratios, so use your cc troops and request as much as possible.
  • Only donate requested troops. Please don't donate if you have less than level 5 archers.
  • We use Warmatch to call bases for war. If you take someone else's target we will be very unhappy. Always check before you attack.
  • ALWAYS use both attacks.
  • No troll bases. Seriously don't do it.
  • If you're TH9 and TH8, always go for three stars. If you think you can't 3 star, then attack a lower base that you can 3 star.
  • Never go for loot in war. We will kick you. If there are no viable targets, practice a three star attack on an already 3 starred base.
  • Especially if you're a new member, ask for help with your attack.
  • If you fail an attack, be receptive to criticism. Don't leave immediately after a failed attack.
  • We use GroupMe chats as well so it's strongly recommended that you make an account there.

Using Warmatch

Getting Started

  1. Go to Warmatch.us
  2. Login or create an account if you don't already have one
  3. Fill out your profile information
  4. Search "Dusk Dominion" to join our (#2YYVCU8C)

Calling Bases through the Website

  1. Go to the dashboard
  2. Click on the current war
  3. Choose the enemy base to you want to call

Using Warmatch through GroupMe

  1. Join our War Room Chat on GM
  2. Use "!wm" to start a command
  3. Call a base or log your attacks and defenses using the command list
